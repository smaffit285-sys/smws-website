# SMWS Landing Site

This package is configured for the custom domain `swms.work` using the included `CNAME` file.

Pages:
- `index.html`
- `services.html`
- `packages.html`
- `process.html`
- `principles.html`
- `faq.html`
- `contact.html`

Logo assets:
- `assets/smws-logo.png` — cropped transparent version of the user-provided logo
- `assets/smws-logo-original.png` — original attached logo image

Before launch, connect `contact.html` to a real form endpoint or booking link.
